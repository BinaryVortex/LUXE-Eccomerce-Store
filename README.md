# 🛒 LUXE - Modern E-commerce Store

A modern, high-performance E-commerce UI built with React, Vite, and Tailwind CSS. This project features a clean, professional design inspired by high-end retail experiences.

![Store Preview](./Screenshot%202025-12-14%20222754.png)

## ✨ Features

-   **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
-   **Product Discovery:** Browse through new arrivals and top-rated products.
-   **Dynamic Filtering:** Easily filter products by category, price, and more.
-   **Smooth Navigation:** Seamless page transitions using modern routing.
-   **Interactive UI:** High-quality components powered by Radix UI and Shadcn.
-   **Shopping Experience:** Complete user flow from product listing to checkout.

## 🚀 Tech Stack

-   **Frontend:** React 18, Vite
-   **Styling:** Tailwind CSS, PostCSS
-   **UI Components:** Radix UI, Shadcn/UI, Lucide React
-   **State Management:** Modern hooks and context-based data flow
-   **Animations:** Framer Motion (via `motion`)

## 🛠️ Getting Started

### Prerequisites

-   Node.js (v18.0 or higher)
-   npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/degimen-ecommerce-store.git
    cd DEGIMEN-Eccomerce-Store-master
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```

## 📂 Project Structure

-   `src/app/components`: Reusable UI components (Navbar, Footer, Product Cards).
-   `src/app/pages`: Main page views (Home, Product Detail, Cart, Checkout).
-   `src/app/data`: Mock data for products and categories.
-   `src/styles`: Global styles and theme configurations.
